# 📝 To-Do List Manager in C

A simple **menu-driven To-Do List Manager** built using **C Programming**. This project helps users organize daily tasks through an easy-to-use console interface.

## ✨ Features

- ➕ Add New Task
- 📋 View Tasks
- ✅ Mark Task as Completed
- 🗑️ Delete Task
- 🎯 Menu-Driven Interface
- 💻 Beginner-Friendly

## 🛠️ Technologies Used

- C Programming
- Standard C Library

## 📚 Concepts Used

- Functions
- Arrays
- Strings
- Loops
- Conditional Statements
- Menu-Driven Programming
